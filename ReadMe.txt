###### I've added the Swagger dependency to speed up the API exploration and testing during development phase.
## To explore the APIs over Swagger UI visit
###### http://localhost:8080/swagger-ui.html

## Lombok
###### To explore and run in IDE please install Lombok plug-in (I've used this lib in this project to speedup the development)
###### If you don't install it you will get see compile time issues in IDE

## Using in memory H2 DB for this project
###### URL to access the DB once the application bootstrapped http://localhost:8080/h2-console

## IDE
###### please import the project in favorite IDE as Maven project (Note: do not forget to install Lombok plugin in your IDE)

## To unit test the application
###### mvn test

## To bootstrap the application
###### mvn spring-boot:run
